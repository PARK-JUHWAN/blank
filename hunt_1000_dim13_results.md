#
# 단, 모든 6/6은 동일 게이트 3개를 통과해야 성립:
#   (1) 자본 — 길목/권리를 *매입·선점*해야(취득위반은 과거로 매몰).
#   (2) 독점등급 — 진짜 monopoly chokepoint여야(희소→5/6, 범용→3~4/6로 즉시 하락).
#   (3) inbound 징수 — buyer가 필연으로 찾아와 자발 지불(집행노동 들면 #1/#5 붕괴).
#
# ★ 6/6 아키타입(자연스러운 대표 길목):
#   1) 지리적 통행 지역권 @ 산악 단일 통과회랑 — 지리=발견필연, 통과자=현존 buyer
#   2) 영구 양륙점/랜딩 권리 @ 해저케이블 단일 양륙점
#   3) 전략입지 정박/슬롯 @ 단일 항만 진입수로
#   4) 임차 스펙트럼 @ (독점) 필수 대역
#   5) 비탄력 로열티 스트림 매입 @ 표준치료 독점신약
#   6) 불가피 기반특허 inbound @ 유일 호환 필수특허 기술
#   7) 식별자/주소 블록 서브리스 @ 희소 레인지(독점보유분)
#   8) 필수규격 IP 영구로열티 매입 @ 단일 호환규격
